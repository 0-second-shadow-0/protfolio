# SECOND SHADOW — Portfolio

Interactive 80s arcade-style space portfolio intro with canvas-based starfield, brown dwarf discovery, and embedded mini-games.

## How it works

- Steer with your mouse through a twinkling starfield
- Discover a brown dwarf → triggers project card (Email Writer Assistant)
- Click the floating snake or dwarf to destroy it → triggers Ink App card
- Click the Go gopher mascot → reveals ultra-bDB card with embedded tic-tac-toe

## Projects

1. **Email Writer Assistant** — API-powered email drafting tool with tone detection and prompt chaining
2. **Ink App** — Low-latency Windows Ink alternative in Python with pressure sensitivity
3. **ultra-bDB** — Embedded Go database engine with ACID transactions and custom B-tree

## Stack

Vanilla HTML, CSS, JavaScript. No frameworks or build tools.

## Fonts

Press Start 2P (Google Fonts) for the retro arcade aesthetic.
